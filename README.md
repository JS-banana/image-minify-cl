# imagemin

## 目标

- 命令行执行，对整个文件夹内的所有图片进行压缩
- 可配置参数
  - 输入、输出path
  - jpeg、png
  - 压缩算法、压缩质量
  - 日志打印

## 待解决

- 输入多层嵌套文件夹结构，输出全部到一级目录下

## 命令

- -v --version ：查看版本信息
- start : 启动压缩
